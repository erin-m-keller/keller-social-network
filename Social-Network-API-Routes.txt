Social Network API

USERS:
- GET: Find all users*
- GET: Find user by ID*
- POST: create user*
- PUT: update user*
- DELETE: delete user*

FRIENDS:
 - POST: add friend*
 - DELETE: remove friend*

REACTIONS:
 - POST: create reaction
 - DELETE: remove reaction

THOUGHTS:
 - GET: get all thoughts*
 - GET: get thought by id
 - POST: create thought*
 - PUT: update thought by id
 - DELETE: delete a thought*
